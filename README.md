# MyProject

This is a Java web application built using JSP and Servlets.

## Features
- Form handling using JSP
- GET and POST methods using Servlets
- Basic MVC structure

## Technologies Used
- Java
- JSP
- Servlet
- Apache Tomcat

## How to Run
1. Import the project into Eclipse / VS Code
2. Configure Apache Tomcat
3. Run the project on the server
4. Open browser and access index.jsp
